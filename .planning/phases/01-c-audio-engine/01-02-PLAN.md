**Create `native/src/faust_wrapper.cpp`** — C++ implementation:

The FaustDSP struct wraps a Faust-generated `dsp` subclass instance. The key design:

1. **Include all 7 generated DSP headers** at the top:
   ```cpp
   #include "sine_dsp.cpp"      // defines class SineDSP : public dsp
   #include "triangle_dsp.cpp"  // defines class TriangleDSP : public dsp
   // ... etc for all 7
   ```
   Note: These are `.cpp` files from Faust codegen but are included as headers because each defines a class. This is the standard Faust embedding pattern.

2. **FaustDSP struct definition:**
   ```cpp
   struct FaustDSP {
       dsp*         instance;     // pointer to Faust dsp base class
       WaveformType type;
       bool         in_use;       // pool management flag
   };
   ```

3. **DSP instance pool:** Pre-allocate 64 `FaustDSP` entries at init. For each entry, create one `dsp` subclass instance using `new SineDSP()` etc. Use a simple array with linear scan for acquire/release (pool is small, only touched from audio thread for release and from caller thread for acquire — use atomic `in_use` flag).

   Actually, reconsider: acquire happens via SPSC message processing ON the audio thread (the MSG_VOICE_ADD handler calls acquire). Release also happens on the audio thread (after crossfade completes or voice removed). So pool access is single-threaded (audio thread only). No atomic needed for pool management — just a plain bool.

4. **Parameter setting:** Use Faust's `APIUI` class to discover parameter paths at init time, then build a fast lookup from short name ("freq") to parameter index. Store the APIUI per DSP instance.

   Simpler alternative: Since we control the .dsp files and know the parameter names, use `dsp->setParamValue(path, value)` with the Faust JSONUI or just iterate `dsp->getNumInputs()` etc. Actually simplest: use the Faust `MapUI` class which maps parameter names to zones (float pointers). Include `faust/gui/MapUI.h` from the Faust include directory. Then:
   ```cpp
   MapUI mapui;
   instance->buildUserInterface(&mapui);
   // Later:
   mapui.setParamValue("freq", 440.0f);  // sets via the mapped zone pointer
   ```

   Store a `MapUI` per FaustDSP instance.

   Updated struct:
   ```cpp
   struct FaustDSP {
       dsp*         instance;
       MapUI        ui;           // parameter name -> zone mapping
       WaveformType type;
       bool         in_use;
   };
   ```

5. **Factory function** to create the right DSP subclass by type:
   ```cpp
   static dsp* create_dsp_instance(WaveformType type) {
       switch (type) {
           case WAVEFORM_SINE:     return new SineDSP();
           case WAVEFORM_TRIANGLE: return new TriangleDSP();
           case WAVEFORM_SAW:      return new SawDSP();
           case WAVEFORM_SQUARE:   return new SquareDSP();
           case WAVEFORM_PULSE:    return new PulseDSP();
           case WAVEFORM_NOISE:    return new NoiseDSP();
           case WAVEFORM_FM:       return new FmDSP();
           default:                return nullptr;
       }
   }
   ```

6. **`faust_wrapper_init`:** Allocate 64 FaustDSP structs. For initial pool, create instances of each type (distribute roughly: 9-10 of each type, or simpler: all 64 as sine initially and re-create on acquire if wrong type). Better approach: **lazy type assignment.** Each pool slot starts with `instance = nullptr, in_use = false`. On `faust_wrapper_acquire(type)`, find a free slot, create the right DSP instance if slot is empty or wrong type, call `instance->init(sample_rate)`, build UI, mark in_use. On `faust_wrapper_release`, mark `in_use = false` but keep the instance alive for reuse.

   This avoids pre-creating 64 instances of unknown types. The only allocation happens on the first acquire of each slot — after that, the instance persists and is reused. Since acquire only happens on the audio thread via SPSC message processing, and we expect at most 32 voices active, the first 32 acquires will allocate (during voice creation, which is user-initiated and infrequent). Subsequent acquires for crossfade may need a different type — in that case, delete old instance and create new.

   **CRITICAL CONCERN:** This means `new`/`delete` can happen on the audio thread during voice creation and waveform changes. This violates the no-malloc-on-audio-thread rule (Pitfall 1).

   Better solution: Pre-allocate ALL 64 instances at init time. Use 9 of each type (9*7=63, +1 extra sine = 64). On acquire, find a free slot matching the requested type. If no free slot of that type exists, repurpose a different-type slot — but this requires delete+new on audio thread.

   **Best solution for v1:** Pre-allocate 2 instances per type per voice would be 32*2*7 = 448, too many. Instead: pre-allocate one pool per waveform type, each with 10 instances (70 total). acquire(type) pulls from that type's pool. If pool exhausted, return NULL (should not happen with 32 max voices). Release returns to the correct type pool. This is zero-allocation on the audio thread.

   Implement as:
   ```cpp
   #define POOL_PER_TYPE 10
   static FaustDSP g_pool[NUM_WAVEFORM_TYPES][POOL_PER_TYPE];
   static int g_sample_rate;
   ```

   `faust_wrapper_init`: for each type, for each slot, `new` the right DSP class, call `init(sample_rate)`, build MapUI, set `in_use = false`.

   `faust_wrapper_acquire(type)`: linear scan g_pool[type] for `in_use == false`, set `in_use = true`, re-init the instance (`instance->instanceInit(sample_rate)` to reset state without reallocation), return pointer.

   `faust_wrapper_release(dsp)`: set `dsp->in_use = false`.

7. **`faust_wrapper_compute`:** Call `instance->compute(frame_count, nullptr, &output_ptr)` where output_ptr is a float* array of 1 element (mono). Faust compute signature is `compute(int count, FAUSTFLOAT** inputs, FAUSTFLOAT** outputs)`.
   ```cpp
   void faust_wrapper_compute(FaustDSP* dsp, int frame_count, float* output) {
       float* outputs[1] = { output };
       dsp->instance->compute(frame_count, nullptr, outputs);
   }
   ```

8. **`faust_wrapper_set_param`:**
   ```cpp
   void faust_wrapper_set_param(FaustDSP* dsp, const char* param_name, float value) {
       dsp->ui.setParamValue(param_name, value);
   }
   ```

**2. apply_control_message — runs on audio thread, handles all SPSC message types:**
```cpp
static void apply_control_message(AudioEngine* eng, const ControlMessage* msg) {
    switch (msg->type) {
        case MSG_VOICE_ADD: {
            // Find free slot
            for (int i = 0; i < MAX_VOICES; i++) {
                if (atomic_load(&eng->voices[i].state) == VOICE_FREE) {
                    VoiceSlot* slot = &eng->voices[i];
                    slot->dsp = faust_wrapper_acquire((WaveformType)msg->voice_add.waveform_type);
                    if (!slot->dsp) break; // pool exhausted
                    slot->waveform_type = msg->voice_add.waveform_type;
                    slot->frequency = msg->voice_add.frequency;
                    slot->amplitude = msg->voice_add.amplitude;
                    slot->pan = 0.0f;
                    slot->detune_cents = 0.0f;
                    slot->attack_time = 0.05f;
                    slot->release_time = 10.0f;
                    slot->dsp_pending = NULL;
                    slot->crossfade_samples_remaining = 0;
                    // Set initial DSP parameters
                    faust_wrapper_set_param(slot->dsp, "freq", slot->frequency);
                    faust_wrapper_set_param(slot->dsp, "amp", slot->amplitude);
                    faust_wrapper_set_param(slot->dsp, "attack", slot->attack_time);
                    faust_wrapper_set_param(slot->dsp, "release", slot->release_time);
                    faust_wrapper_set_param(slot->dsp, "gate", 1.0f); // trigger envelope
                    atomic_store(&slot->state, VOICE_ACTIVE);
                    atomic_fetch_add(&eng->active_voice_count, 1);
                    break;
                }
            }
            break;
        }
        case MSG_VOICE_REMOVE: {
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            // Release gate — envelope handles fade-out
            faust_wrapper_set_param(slot->dsp, "gate", 0.0f);
            // Mark free after release time — for now, immediately free
            // (sleep-before-destroy is a UI/Dart concern in Phase 3)
            faust_wrapper_release(slot->dsp);
            slot->dsp = NULL;
            if (slot->dsp_pending) {
                faust_wrapper_release(slot->dsp_pending);
                slot->dsp_pending = NULL;
            }
            atomic_store(&slot->state, VOICE_FREE);
            atomic_fetch_sub(&eng->active_voice_count, 1);
            break;
        }
        case MSG_SET_FREQUENCY: {
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            slot->frequency = msg->float_value;
            faust_wrapper_set_param(slot->dsp, "freq", slot->frequency);
            break;
        }
        case MSG_SET_AMPLITUDE: {
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            slot->amplitude = msg->float_value;
            faust_wrapper_set_param(slot->dsp, "amp", slot->amplitude);
            break;
        }
        case MSG_SET_PAN: {
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            slot->pan = msg->float_value; // pan applied in mix loop, not in DSP
            break;
        }
        case MSG_SET_DETUNE: {
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            slot->detune_cents = msg->float_value;
            faust_wrapper_set_param(slot->dsp, "detune", slot->detune_cents);
            break;
        }
        case MSG_SET_WAVEFORM: {
            // Waveform crossfade per D-02: 20ms overlap
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            if (slot->waveform_type == msg->int_value) break; // same type, no-op

            FaustDSP* new_dsp = faust_wrapper_acquire((WaveformType)msg->int_value);
            if (!new_dsp) break; // pool exhausted

            // Copy current parameters to new DSP
            faust_wrapper_set_param(new_dsp, "freq", slot->frequency);
            faust_wrapper_set_param(new_dsp, "amp", slot->amplitude);
            faust_wrapper_set_param(new_dsp, "detune", slot->detune_cents);
            faust_wrapper_set_param(new_dsp, "attack", slot->attack_time);
            faust_wrapper_set_param(new_dsp, "release", slot->release_time);
            faust_wrapper_set_param(new_dsp, "gate", 1.0f);

            slot->dsp_pending = new_dsp;
            slot->crossfade_samples_remaining = (int)(0.020f * eng->sample_rate); // 960 at 48kHz
            slot->crossfade_gain = 0.0f;
            slot->waveform_type = msg->int_value;
            atomic_store(&slot->state, VOICE_FADING);
            break;
        }
        case MSG_SET_GATE: {
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            faust_wrapper_set_param(slot->dsp, "gate", (float)msg->int_value);
            break;
        }
        case MSG_SET_GATE_TIMES: {
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            slot->attack_time = msg->gate_times.attack_s;
            slot->release_time = msg->gate_times.release_s;
            faust_wrapper_set_param(slot->dsp, "attack", slot->attack_time);
            faust_wrapper_set_param(slot->dsp, "release", slot->release_time);
            break;
        }
        case MSG_SET_MOD_RATIO: {
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            faust_wrapper_set_param(slot->dsp, "mod_ratio", msg->float_value);
            break;
        }
        case MSG_SET_MOD_INDEX: {
            int id = msg->voice_id;
            if (id < 0 || id >= MAX_VOICES) break;
            VoiceSlot* slot = &eng->voices[id];
            if (atomic_load(&slot->state) == VOICE_FREE) break;
            faust_wrapper_set_param(slot->dsp, "mod_index", msg->float_value);
            break;
        }
        case MSG_SET_MASTER_VOLUME: {
            eng->master_volume = msg->float_value;
            break;
        }
    }
```

**3. audio_callback — the heart of the engine:**
```cpp
static void audio_callback(ma_device* device, void* output,
                            const void* input, ma_uint32 frame_count) {
    (void)input;
    AudioEngine* eng = (AudioEngine*)device->pUserData;
    float* out = (float*)output;

    // 1. Panic check — FIRST, before anything else (per Pitfall 3)
    if (atomic_load_explicit(&eng->is_silent, memory_order_relaxed)) {
        memset(out, 0, frame_count * 2 * sizeof(float));
        return;
    }

    // 2. Drain SPSC control queue
    ControlMessage msg;
    while (eng->control_queue->try_dequeue(msg)) {
        apply_control_message(eng, &msg);
    }

    // 3. Clear output buffer
    memset(out, 0, frame_count * 2 * sizeof(float));

    // 4. Temp buffer for per-voice mono output
    float tmp[MAX_BUFFER_SIZE];

    // 5. Mix active voices
    for (int i = 0; i < MAX_VOICES; i++) {
        int state = atomic_load_explicit(&eng->voices[i].state, memory_order_relaxed);
        if (state == VOICE_FREE) continue;

        VoiceSlot* slot = &eng->voices[i];

        if (state == VOICE_FADING && slot->dsp_pending) {
            // Crossfade: compute both old and new DSP
            float tmp_old[MAX_BUFFER_SIZE];
            float tmp_new[MAX_BUFFER_SIZE];
            faust_wrapper_compute(slot->dsp, (int)frame_count, tmp_old);
            faust_wrapper_compute(slot->dsp_pending, (int)frame_count, tmp_new);

            int xfade_remaining = slot->crossfade_samples_remaining;
            int xfade_total = (int)(0.020f * eng->sample_rate);

            for (ma_uint32 s = 0; s < frame_count; s++) {
                float progress;
                if (xfade_remaining <= 0) {
                    progress = 1.0f;
                } else {
                    progress = 1.0f - ((float)xfade_remaining / (float)xfade_total);
                    xfade_remaining--;
                }
                float sample = tmp_old[s] * (1.0f - progress) + tmp_new[s] * progress;

                float amp_L = slot->amplitude * (1.0f - slot->pan) * 0.5f;
                float amp_R = slot->amplitude * (1.0f + slot->pan) * 0.5f;
                out[s*2]   += sample * amp_L * eng->master_volume;
                out[s*2+1] += sample * amp_R * eng->master_volume;
            }

            slot->crossfade_samples_remaining = xfade_remaining;
            if (xfade_remaining <= 0) {
                // Crossfade complete — swap and release old
                faust_wrapper_release(slot->dsp);
                slot->dsp = slot->dsp_pending;
                slot->dsp_pending = NULL;
                atomic_store(&slot->state, VOICE_ACTIVE);
            }
        } else {
            // Normal: compute single DSP
            faust_wrapper_compute(slot->dsp, (int)frame_count, tmp);

            float amp_L = slot->amplitude * (1.0f - slot->pan) * 0.5f;
            float amp_R = slot->amplitude * (1.0f + slot->pan) * 0.5f;
            for (ma_uint32 s = 0; s < frame_count; s++) {
                out[s*2]   += tmp[s] * amp_L * eng->master_volume;
                out[s*2+1] += tmp[s] * amp_R * eng->master_volume;
            }
        }
    }
}
```

```cpp
extern "C" int justifier_init(int sample_rate, int buffer_size) {
    memset(&g_engine, 0, sizeof(g_engine));
    g_engine.sample_rate = sample_rate;
    g_engine.buffer_size = buffer_size;
    g_engine.master_volume = 1.0f;
    atomic_store(&g_engine.is_silent, false);
    atomic_store(&g_engine.xrun_count, 0);
    atomic_store(&g_engine.active_voice_count, 0);

    // Init all voice slots as free
    for (int i = 0; i < MAX_VOICES; i++) {
        atomic_store(&g_engine.voices[i].state, VOICE_FREE);
        g_engine.voices[i].dsp = NULL;
        g_engine.voices[i].dsp_pending = NULL;
    }

    // Init Faust DSP pool
    if (faust_wrapper_init(sample_rate) != 0) return -1;

    // Create SPSC queue with capacity 1024 messages
    g_engine.control_queue = new moodycamel::ReaderWriterQueue<ControlMessage>(1024);

    // Init miniaudio device (per D-03, D-04, D-09)
    ma_device_config config = ma_device_config_init(ma_device_type_playback);
    config.playback.format   = ma_format_f32;
    config.playback.channels = 2;
    config.sampleRate        = (ma_uint32)sample_rate;
    config.periodSizeInFrames = (ma_uint32)buffer_size; // hint — PipeWire may override per D-04
    config.dataCallback      = audio_callback;
    config.pUserData         = &g_engine;

    if (ma_device_init(NULL, &config, &g_engine.device) != MA_SUCCESS) {
        faust_wrapper_shutdown();
        delete g_engine.control_queue;
        return -2;
    }

    if (ma_device_start(&g_engine.device) != MA_SUCCESS) {
        ma_device_uninit(&g_engine.device);
        faust_wrapper_shutdown();
        delete g_engine.control_queue;
        return -3;
    }

    g_engine.running = true;
    return 0;
}
```

```cpp
    ma_device_uninit(&g_engine.device);
```

**6. Voice management API** — each function enqueues a ControlMessage:

`justifier_voice_add`: Build MSG_VOICE_ADD message, enqueue. Return the voice_id (slot index). To determine which slot will be used, scan for the first free slot from the caller side and use that as the voice_id. Note: there's a race condition if two adds happen before the audio thread processes them. Solve by using an atomic `next_voice_id` counter — NO. Simpler: the caller (UI thread) does a scan of `voices[i].state` to find a free slot. Since voice creation is infrequent and user-initiated, the scan is fine. If the slot gets claimed between scan and audio-thread processing, the audio thread's handler will skip it (already active).

Actually, the cleaner pattern: `justifier_voice_add` scans for a free slot, marks it as "pending" with a separate atomic state (or just reserves it by setting state to VOICE_ACTIVE from the caller side), then enqueues the MSG_VOICE_ADD with the slot index. The audio thread handler initializes the DSP. But modifying state from the caller thread while the audio thread reads it requires care.

**Simplest safe pattern:** Use an atomic counter. `justifier_voice_add` atomically increments a counter, uses counter % MAX_VOICES as a starting search index, linearly scans for VOICE_FREE, returns the index. The audio thread does the actual initialization via the queued message. If no free slot found, return -1.

```cpp
static _Atomic int g_voice_id_counter = 0;

extern "C" int justifier_voice_add(WaveformType type, float frequency, float amplitude) {
    // Find a free slot (approximate — audio thread does final check)
    int start = atomic_fetch_add(&g_voice_id_counter, 1) % MAX_VOICES;
    for (int i = 0; i < MAX_VOICES; i++) {
        int idx = (start + i) % MAX_VOICES;
        if (atomic_load(&g_engine.voices[idx].state) == VOICE_FREE) {
            ControlMessage msg = {};
            msg.type = MSG_VOICE_ADD;
            msg.voice_id = idx;
            msg.voice_add.waveform_type = (int)type;
            msg.voice_add.frequency = frequency;
            msg.voice_add.amplitude = amplitude;
            g_engine.control_queue->enqueue(msg);
            return idx;
        }
    }
    return -1; // all slots full
}
```
