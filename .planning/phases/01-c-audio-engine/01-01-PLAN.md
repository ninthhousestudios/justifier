**Step 4: Create all 7 Faust .dsp files.**

Each DSP file MUST follow this convention per D-05 and D-06:
- Flat parameter names using `nentry` (not `hslider`/`vslider` — see Pitfall in RESEARCH.md)
- Parameters: `freq` (Hz), `amp` (0.0-1.0), `gate` (0/1), `pan` (-1.0 to 1.0), `detune` (cents)
- FM additionally: `mod_ratio`, `mod_index` (per D-06)
- Use `si.smooth(0.999)` on freq/amp/pan for zipper-noise-free parameter changes
- Gate with configurable attack/release envelope: attack default 0.05s, release default 10.0s
- Output is MONO (the C wrapper handles stereo panning per D-05)

**sine.dsp:**
```faust
import("stdfaust.lib");
freq = nentry("freq", 440.0, 20.0, 20000.0, 0.01) : si.smoo;
amp = nentry("amp", 0.0, 0.0, 1.0, 0.001) : si.smoo;
gate = nentry("gate", 0, 0, 1, 1);
attack = nentry("attack", 0.05, 0.001, 1.0, 0.001);
release = nentry("release", 10.0, 0.01, 30.0, 0.01);
detune = nentry("detune", 0.0, -100.0, 100.0, 0.01);
effective_freq = freq * 2.0^(detune / 1200.0);
envelope = en.are(attack, release, gate);
process = os.osc(effective_freq) * amp * envelope;
```

**triangle.dsp:** Same structure, replace `os.osc` with `os.triangle`.

**saw.dsp:** Same structure, replace `os.osc` with `os.sawtooth`.

**square.dsp:** Same structure, replace `os.osc` with `os.square`.

**pulse.dsp:** Same structure, replace `os.osc` with `os.pulsetrain` (use `os.lf_pulsetrain` if `os.pulsetrain` is unavailable — check Faust docs; may need duty cycle parameter `nentry("duty", 0.5, 0.01, 0.99, 0.01)`).

**noise.dsp:** Same structure but replace oscillator with `no.noise`. No freq or detune parameters needed (noise is broadband), but keep them in the interface for API uniformity (just ignore freq in the computation).

**fm.dsp:** Per D-06, add `mod_ratio` and `mod_index`:
```faust
import("stdfaust.lib");
freq = nentry("freq", 440.0, 20.0, 20000.0, 0.01) : si.smoo;
amp = nentry("amp", 0.0, 0.0, 1.0, 0.001) : si.smoo;
gate = nentry("gate", 0, 0, 1, 1);
attack = nentry("attack", 0.05, 0.001, 1.0, 0.001);
release = nentry("release", 10.0, 0.01, 30.0, 0.01);
detune = nentry("detune", 0.0, -100.0, 100.0, 0.01);
mod_ratio = nentry("mod_ratio", 1.0, 0.0, 20.0, 0.01) : si.smoo;
mod_index = nentry("mod_index", 0.0, 0.0, 25.0, 0.01) : si.smoo;
effective_freq = freq * 2.0^(detune / 1200.0);
modulator = os.osc(effective_freq * mod_ratio) * mod_index * effective_freq;
carrier = os.osc(effective_freq + modulator);
envelope = en.are(attack, release, gate);
process = carrier * amp * envelope;
```

**Faust codegen via add_custom_command** (per RESEARCH.md Pattern 4):
- Input: `native/dsp/{name}.dsp`
- Output: `${CMAKE_CURRENT_BINARY_DIR}/generated/{name}_dsp.cpp`
- Command: `faust -lang cpp -cn {Name}DSP -i "${FAUST_DSP_DIR}/{name}.dsp" -o "${FAUST_GEN_DIR}/{name}_dsp.cpp"`
- Use `-cn {Name}DSP` to give each generated class a unique name (SineDSP, TriangleDSP, etc.) instead of the default `mydsp`
- Use `-i` flag to inline library code (avoids runtime library path issues)
- Do NOT use `-a minimal.cpp` — instead, include the Faust `dsp/dsp.h` and `gui/UI.h` headers directly from the Faust include directory. The `-a` architecture file adds `main()` and other scaffolding we don't want.
- Loop over: sine, triangle, saw, square, pulse, noise, fm

**Library target:**
```cmake
set(FAUST_DSP_FILES sine triangle saw square pulse noise fm)
# ... (codegen loop produces GENERATED_DSP_SOURCES)

add_library(justifier_audio STATIC
    src/audio_engine.c
    src/faust_wrapper.cpp
    ${GENERATED_DSP_SOURCES}
)
target_include_directories(justifier_audio PRIVATE
    ${CMAKE_CURRENT_SOURCE_DIR}/include
    ${CMAKE_CURRENT_SOURCE_DIR}/src
    ${FAUST_GEN_DIR}
    ${FAUST_INCLUDE_DIR}
)
target_link_libraries(justifier_audio PRIVATE
    pthread
    m
)
# Linux: link ALSA for miniaudio backend
if(UNIX AND NOT APPLE)
    target_link_libraries(justifier_audio PRIVATE asound dl)
endif()
# macOS: link CoreAudio frameworks
if(APPLE)
    target_link_libraries(justifier_audio PRIVATE
        "-framework CoreAudio"
        "-framework AudioToolbox"
        "-framework CoreFoundation"
    )
endif()
```
